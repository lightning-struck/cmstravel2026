# Система управления контентом "Мир Путешествий"

Дипломный проект: админ-панель на Strapi + клиентский сайт на React для туристического агентства.

## Структура

- `/frontend` — React-сайт с турами
- `/backend` — Админ-панель и API на Strapi

## Запуск проекта

### 1. Бэкенд (Strapi)

```bash
cd backend
npm install   # только при первом запуске
npm run develop

### 2. Фронтенд

cd frontend
npm install   # только при первом запуске
npm start
